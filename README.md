# yandex-afisha-loyalty-analysis



\# Анализ лояльности пользователей Яндекс Афиши



\## Описание проекта

Анализ поведения пользователей Яндекс Афиши для выявления факторов, влияющих на повторные покупки.



\## Структура проекта

\- `analysis.ipynb` - основной ноутбук с анализом

\- `requirements.txt` - зависимости проекта

\- `.gitignore` - игнорируемые файлы



\## Установка и запуск

1\. Установите зависимости: `pip install -r requirements.txt`

2\. Запустите Jupyter: `jupyter notebook`

3\. Откройте `analysis.ipynb`
